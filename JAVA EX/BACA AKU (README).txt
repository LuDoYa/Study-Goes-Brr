Oke, di sini aku, Nathan, akan ajarin kalian bagaimana caranya untuk melihat dan mengakses file2 yang buanyak
di sini yang sudah dikemas dalam .zip).

1. Pastikan kalian punya WinRAR di laptop kalian.
   WinRAR merupakan aplikasi cukup penting untuk mengekstraksi file-file yang dikompres seperti file ini.
   Jika belum diinstal, silakan kunjungi (copas linknya) :

	 https://www.win-rar.com/start.html?&L=0

2. Pertama, download .zip file-nya dulu dari laptop via WhatsApp (bisa lewat WhatsApp Web atau aplikasi, kalo udah diinstal)
   Kemudian, klik "Open" atau "Buka" yang berwarna kuning di POJOK KIRI BAWAH layar laptop kalian.

3. Pada saat kalian buka zip-nya, di tampilan aplikasi WinRAR ada icon/logo/gambar untuk "Extract To" atau "Ekstrakkan ke"
   atau terjemahan yang bermakna sama persis. Klik icon itu.

4. Di bawah kotak "Destination path" ada kotak berisikan peta untuk directory (lokasi) penyimpanan hasil ekstraksi filenya.
   Silakan untuk dipindah ke lokasi lain jika kalian ingin disimpan di lokasi yang berbeda dari yang sudah diatur.

5. Klik "OK" dan tunggu hasil ekstrasi.

6. Di File Explorer/Library, cari folder penyimpanan hasil ekstraksi tadi.

7. Tinggal klak klik foldernya, di dalamnya sudah ada file Java yang sudah disiapkan.

8. Untuk menjalankan, bisa kalian gunakan cmd dengan kode "cd" ke arah subfolder2 yang bermuara ke folder penyimpanannya.
   Lalu ketik "java <file_name>.java" -> ganti <file_name> dengan nama file yang bersangkutan.

	Alternatifnya, kalian bisa buka lewat NetBeans, lalu klik File (pojok kanan atas) -> Open File dan arahkan ke lokasi
	penyimpanan. Lalu dapat dijalankan seperti biasa.

	Secara dasarnya, kalian dapat buka di notepad untuk mempelajari semua syntax-nya.

	Untuk pengguna VSCode, klik dua kali filenya dan jalankan seperti biasa.

9. Tadaa! Selesai!
